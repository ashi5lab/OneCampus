# Cohorts Feature (Frontend)

**Purpose**: List and create cohorts (Class/Course Section/Playgroup, vocabulary-driven). Mirrors the learners feature's structure.

**API Endpoints used**: `GET /api/v1/cohorts`, `POST /api/v1/cohorts`.

**Known limitation**: `unit_id` is a raw numeric field for now — no unit picker UI yet, since units management itself isn't built out on the frontend.
