# Q School Design System

## Vision
Create a clean, modern, professional school management UI inspired by Linear, Notion, Stripe Dashboard and Material Design 3.

## Tech Stack
- React
- Tailwind CSS
- shadcn/ui
- Lucide Icons
- Framer Motion

## Theme Tokens
### Colors
- Background: `#F6F8FC`
- Surface: `#FFFFFF`
- Primary: `#4F46E5`
- Primary Hover: `#4338CA`
- Border: `#E5E7EB`
- Text Primary: `#111827`
- Text Secondary: `#374151`
- Muted: `#6B7280`
- Success: `#16A34A`
- Warning: `#F59E0B`
- Danger: `#DC2626`
- Info: `#2563EB`

Use CSS variables only. Never hardcode colors.

## Typography
- Font: Inter (preferred)
- 8px spacing system
- Cards: 16px radius
- Buttons/Inputs: 12px radius

## Global Rules
- One sidebar throughout the application.
- One page header pattern.
- One table component.
- One card component.
- One button component.
- Maximum 5 summary cards per page.
- Use skeleton loaders.
- Support light/dark themes and color themes using CSS variables.
